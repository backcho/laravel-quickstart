# 02. 프로젝트 설치/생성

> 목표: Laravel 프로젝트 생성 + 초기 설정

---

## 프로젝트 생성

```bash
# 프로젝트 폴더로 이동
cd ~/Herd        # Mac (Herd)
cd ~/Sites       # Mac (Valet)
cd C:\laragon\www  # Windows (Laragon)

# Laravel 프로젝트 생성
composer create-project laravel/laravel my-project

# 프로젝트 폴더로 이동
cd my-project
```

### Laragon 빠른 생성 (Windows)

1. Laragon 우클릭
2. Quick app → Laravel
3. 프로젝트 이름 입력
4. 자동으로 생성 및 브라우저 오픈

---

## 초기 설정

### SQLite 데이터베이스 (기본값)

Laravel 12에서는 SQLite가 기본값입니다. 별도 설정 없이 바로 사용 가능합니다.

```bash
# 마이그레이션 실행
php artisan migrate
```

---

## 실행 확인

### Herd / Laragon 사용 시

별도 서버 실행 없이 브라우저에서:
- **URL**: `http://my-project.test`

### artisan serve 사용 시

```bash
php artisan serve
```
- **URL**: `http://localhost:8000`

브라우저에서 Laravel 기본 Welcome 페이지가 보이면 성공입니다!

---

## 프로젝트 구조

```
my-project/
├── app/                 # 애플리케이션 코드
│   ├── Http/Controllers/ # 컨트롤러
│   └── Models/          # Eloquent 모델
├── database/
│   ├── migrations/      # DB 마이그레이션
│   └── seeders/         # 시드 데이터
├── resources/views/     # Blade 템플릿
├── routes/web.php       # 웹 라우트
└── .env                 # 환경 설정
```
