# 05. 보너스: UI 커스텀

> 목표: AI에게 디자인 요청하기

기능이 완성되었다면, 이제 디자인을 입힐 차례입니다.

---

## Tailwind CSS 기본

laravel/breeze를 설치하면 Tailwind CSS가 기본으로 포함됩니다.

```bash
# 개발 모드 (자동 새로고침)
npm run dev

# 프로덕션 빌드
npm run build
```

---

## AI에게 디자인 요청하기

### 색상 테마 변경

```
Tailwind CSS 색상 테마를 변경해줘.

Primary: #4F46E5 (인디고)
전체적으로 모던하고 깔끔한 느낌으로.
버튼 둥글게, 카드에 그림자 추가해줘.
```

### 네비게이션 커스터마이징

```
네비게이션 바를 다음과 같이 수정해줘:

- 왼쪽: 로고 + 메인 메뉴 (홈, 게시판)
- 오른쪽: 로그인/회원가입 또는 사용자 드롭다운
- 관리자인 경우 "관리자" 메뉴 추가
- 현재 페이지 메뉴 하이라이트
```

### 다크 테마 추가

```
다크 테마를 추가해줘:

1. 토글 버튼으로 라이트/다크 전환
2. 로컬 스토리지에 선택 저장
3. 시스템 설정 자동 감지 (prefers-color-scheme)
```

### 게시판 디자인 개선

```
게시판 목록 페이지를 카드형으로 리디자인해줘:

- 카드형 목록
- 호버 시 애니메이션
- 작성자, 작성일 표시
- 페이지네이션 스타일 개선
```

### 무료 Tailwind 템플릿

- [Tailwind UI](https://tailwindui.com/) - 공식 컴포넌트 (일부 무료)

### 아이콘

```blade

<svg class="w-6 h-6" fill="none" stroke="currentColor" viewBox="0 0 24 24">
    ...
</svg>
```

[Heroicons](https://heroicons.com/) - Tailwind와 찰떡 궁합

```
AI에게 요청:

Pretendard 웹폰트를 적용해줘.
```

## Filament 백오피스 테마

Filament은 기본적으로 깔끔한 UI를 제공합니다. 추가 커스터마이징:

```
AI에게 요청:

Filament 대시보드에 다음 위젯을 추가해줘:
- 최근 7일 가입자 차트
- 최근 7일 게시글 차트
- 인기 게시글 Top 5
```
