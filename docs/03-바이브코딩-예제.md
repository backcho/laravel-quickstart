# 03. 바이브코딩: 게시판 + 백오피스

> 목표: AI에게 복사-붙여넣기로 완성된 서비스 요청하기

이 문서는 **Claude, ChatGPT, Cursor** 등 AI 코딩 도구에 그대로 복사해서 붙여넣을 수 있는 프롬프트를 제공합니다.

---

## 핵심 패키지 3종

| 기능 | 패키지 | 설명 |
|------|--------|------|
| 회원가입/로그인 | `laravel/breeze` | Blade + Tailwind 기반 인증 시스템 |
| 백오피스 (관리자 패널) | `filament/filament` | 강력한 관리자 패널 자동 생성 |
| 권한 관리 (RBAC) | `spatie/laravel-permission` | admin/user 역할 구분 |

이 세 가지 패키지만 알면 **인증 + 게시판 + 백오피스**가 완성됩니다.

---

## 원클릭 프롬프트

### 전체 서비스 한 번에 요청하기

아래 프롬프트를 AI에게 그대로 복사해서 붙여넣으세요:

```
이 프로젝트는 라라벨 12.x가 설치되어 있습니다.
라라벨의 패키지들을 활용해서 게시판을 만들어줘.

[인증]
- laravel/breeze로 회원가입/로그인 구현
- Blade + Tailwind 스택 사용

[게시판]
- 글쓰기는 로그인 필요
- 수정/삭제는 본인만
- 비회원도 읽을 수 있어야 해

[백오피스]
- filament 패키지로 /admin 백오피스 만들어줘
- 회원 관리, 게시물 관리 기능
- 간단한 통계 (전체 회원수, 게시글수, 오늘 가입자)

[권한]
- spatie/laravel-permission으로 역할 관리
- admin 역할이 있는 사람만 백오피스 접근 가능

[기본 계정]
- admin@example.com / password 로 관리자 계정 생성해줘

전체 설치부터 완료까지 단계별로 진행해줘.
```

---

## 단계별 프롬프트 (필요한 것만)

### 1단계: 인증만 먼저

```
laravel/breeze를 설치하고 회원가입/로그인 기능을 구현해줘.
Blade + Tailwind 스택으로 해줘.
```

**AI가 할 일:**
- Breeze 설치 및 설정
- 회원가입, 로그인, 로그아웃 페이지 생성
- Tailwind CSS 설정

---

### 2단계: 게시판 추가

```
Post 모델로 게시판을 만들어줘.

- 목록: 누구나 볼 수 있음, 페이지네이션 10개
- 상세: 누구나 볼 수 있음
- 작성: 로그인한 사용자만
- 수정/삭제: 본인만 가능

Policy를 사용해서 권한 체크해줘.
```

**AI가 할 일:**
- Post 모델 + 마이그레이션
- PostController (리소스 컨트롤러)
- PostPolicy (본인만 수정/삭제)
- Blade 뷰 파일들

---

### 3단계: 권한 시스템

```
spatie/laravel-permission을 설치하고
admin, user 두 가지 역할을 만들어줘.

회원가입하면 자동으로 user 역할이 부여되게 해줘.
```

**AI가 할 일:**
- 패키지 설치 및 설정
- User 모델에 HasRoles 트레이트 추가
- 역할 시더 생성

---

### 4단계: Filament 백오피스

```
filament/filament을 설치하고 /admin 백오피스를 만들어줘.

1. UserResource: 회원 목록, 상세, 역할 변경
2. PostResource: 게시글 목록, 상세, 삭제
3. 대시보드 위젯: 전체 회원수, 게시글수, 오늘 가입자

admin 역할만 접근 가능하게 설정해줘.
```

**AI가 할 일:**
- Filament 설치 및 설정
- UserResource, PostResource 생성
- 대시보드 위젯
- 접근 권한 미들웨어 설정

---

### 5단계: 관리자 계정 생성

```
Seeder로 기본 관리자 계정을 생성해줘.
- 이메일: admin@example.com
- 비밀번호: password
- 역할: admin
```

### 폴더 구조

```
app/
├── Filament/
│   ├── Resources/
│   │   ├── UserResource.php
│   │   └── PostResource.php
```

### URL 구조

| URL | 설명 | 접근 권한 |
|-----|------|----------|
| `/register` | 회원가입 | 비로그인 |
| `/login` | 로그인 | 비로그인 |

| `/admin/users` | 회원 관리 | admin 역할 |
| `/admin/posts` | 게시글 관리 | admin 역할 |

## 패키지별 핵심 포인트

### laravel/breeze

- 인증 기능 + UI 템플릿 자동 생성
- Blade (기본) / Livewire / Inertia (Vue/React) 선택 가능
- Tailwind CSS 기반

### filament/filament

- CRUD 리소스 자동 생성
- 대시보드 위젯 (차트, 통계)
- 내장된 인증/권한 시스템
- 한국어 지원

### spatie/laravel-permission

- 역할(Role) + 권한(Permission) 관리
- Blade 디렉티브: `@role('admin')`, `@can('edit posts')`
- 미들웨어: `role:admin`, `permission:edit posts`

## 트러블슈팅

### "Route [login] not defined" 에러

Breeze 설치 후 마이그레이션을 실행했는지 확인:

```bash
php artisan migrate
```

### Filament 페이지가 안 보임

1. Filament 설정 퍼블리시:
```bash
php artisan filament:install --panels
```

2. 관리자 계정 생성:
```bash
php artisan make:filament-user
```

### 권한이 적용 안 됨

spatie 패키지 캐시 클리어:

```bash
php artisan permission:cache-reset
```
