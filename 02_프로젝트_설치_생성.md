# 02. 프로젝트 설치/생성

> 목표: Laravel 프로젝트 생성 + 초기 설정

---

## 프로젝트 생성

### 방법 1: Composer (권장)

```bash
# 프로젝트 폴더로 이동
cd ~/Herd        # Mac (Herd)
cd ~/Sites       # Mac (Valet)
cd C:\laragon\www  # Windows (Laragon)

# Laravel 프로젝트 생성
composer create-project laravel/laravel my-project

# 프로젝트 폴더로 이동
cd my-project
```

### 방법 2: Laravel Installer

```bash
# Laravel Installer가 설치되어 있다면
laravel new my-project

# 프로젝트 폴더로 이동
cd my-project
```

### Laragon 빠른 생성 (Windows)

Laragon에서는 GUI로 더 빠르게 생성할 수 있습니다:

1. Laragon 우클릭
2. Quick app → Laravel
3. 프로젝트 이름 입력
4. 자동으로 생성 및 브라우저 오픈

## 초기 설정

### .env 파일

프로젝트 루트에 `.env` 파일이 자동 생성됩니다. 최소한의 설정만 확인합니다.

```env
APP_NAME="My Project"
APP_ENV=local
APP_DEBUG=true
APP_URL=http://my-project.test

# 데이터베이스 설정 (아래에서 설명)
DB_CONNECTION=sqlite
```

### SQLite 데이터베이스 사용 (권장)

초기 개발 단계에서는 SQLite를 권장합니다.

**장점:**
- 별도 DB 서버 설치 불필요
- 파일 하나로 관리
- Laravel 11+ 에서는 기본값

**설정 방법:**

```env
# .env 파일에서
DB_CONNECTION=sqlite
# 아래 항목들은 주석 처리하거나 삭제
# DB_HOST=127.0.0.1
# DB_PORT=3306
# DB_DATABASE=laravel
# DB_USERNAME=root
# DB_PASSWORD=
```

Laravel 11 이상에서는 `database/database.sqlite` 파일이 자동 생성됩니다.
Laravel 10 이하라면 수동으로 생성:

```bash
touch database/database.sqlite
```

### 마이그레이션 실행

```bash
php artisan migrate
```

## 실행 확인

### Herd / Laragon 사용 시

별도 서버 실행 없이 바로 브라우저에서 접근 가능합니다:

- **URL**: `http://my-project.test`

### artisan serve 사용 시

Valet 없이 사용하거나, 다른 포트가 필요한 경우:

```bash
php artisan serve
# 또는 포트 지정
php artisan serve --port=8080
```

- **URL**: `http://localhost:8000` (또는 지정한 포트)

### 확인 사항

브라우저에서 Laravel 기본 Welcome 페이지가 보이면 성공입니다!

## Git 초기화

버전 관리를 위해 Git을 초기화합니다.

```bash
git init
git add .
git commit -m "Initial commit: Laravel project created"
```

### .gitignore 확인

Laravel은 기본적으로 적절한 `.gitignore`가 포함되어 있습니다:

```gitignore
/vendor
/node_modules
.env
/database/*.sqlite
# ... 기타
```

## 프로젝트 구조 이해

```
my-project/
├── app/                 # 애플리케이션 코드
│   ├── Http/
│   │   └── Controllers/ # 컨트롤러
│   └── Models/          # Eloquent 모델
├── config/              # 설정 파일
├── database/
│   ├── migrations/      # DB 마이그레이션
│   └── seeders/         # 시드 데이터
├── public/              # 웹 루트 (index.php)
├── resources/
│   └── views/           # Blade 템플릿
├── routes/
│   └── web.php          # 웹 라우트
├── .env                 # 환경 설정
└── composer.json        # PHP 의존성
```

## 유용한 artisan 명령어

```bash
# 라우트 목록 확인
php artisan route:list

# 캐시 클리어
php artisan cache:clear
php artisan config:clear
php artisan view:clear

# 모든 캐시 한번에 클리어
php artisan optimize:clear

# 컨트롤러 생성
php artisan make:controller PostController

# 모델 생성 (마이그레이션 포함)
php artisan make:model Post -m

# 모든 것 한번에 생성 (모델 + 마이그레이션 + 컨트롤러 + 시더 + 팩토리)
php artisan make:model Post -a
```
